🏟️ SportNest
📌 Project Name

SportNest

🎯 Purpose

SportNest is a modern sports venue booking platform that allows users to discover, filter, and book sports facilities such as football turfs, cricket grounds, futsal courts, basketball courts, and more.

It is designed to make sports booking fast, simple, and centralized with a clean UI and real-time booking experience.

👉 https://sport-nest-rouge.vercel.app

## ✨ Features

- 🏟️ Browse sports facilities (football, cricket, futsal, basketball, etc.)
- 🔍 Filter facilities dynamically using URL query params
- 📅 Booking system for sports venues
- 🔐 Authentication system (Better Auth + JWT)
- 🗄️ MongoDB database integration
- ⚡ Next.js App Router for optimized routing
- 🎨 Modern UI with HeroUI + Tailwind CSS
- 🔔 Toast notifications for actions
- 👤 User dashboard (My Bookings, Profile)
- 🛠️ Admin features (Add / Manage Facilities)
- 📱 Fully responsive design (mobile + desktop)
- 🚀 Fast deployment on Vercel

🚀 Getting Started
# Install dependencies
npm install

# Run development server
npm run dev

# Build project
npm run build

# Start production server
npm run start
